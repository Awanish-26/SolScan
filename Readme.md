
# SolScan

A React Native application built with Expo for scanning and analyzing Solana blockchain transactions.

## Project Structure

```
SolScan/
├── App.tsx              # Main application component
├── index.ts             # Entry point
├── css.ts               # Styling utilities
├── app.json             # Expo configuration
├── package.json         # Dependencies
├── tsconfig.json        # TypeScript configuration
├── assets/              # Application assets
└── .expo/               # Expo configuration cache
```

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- Expo CLI (`npm install -g expo-cli`)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on your device using the Expo app or emulator.

- Solana blockchain transaction scanning
